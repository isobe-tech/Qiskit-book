# Chapter 13 code

This folder contains the reproducible code for Chapter 13.

## Files

- `sec13_01_algorithm_skeleton.py`: Generates the circuit figures and histograms for the chapter.
  - Circuit figures: `ch13_circuit_*.png`
  - Ideal probabilities: `ch13_probs_*.png`
  - Sampled counts (optional): `ch13_counts_*.png`

## Run

From the repository root:

```text
python github/code/ch13/sec13_01_algorithm_skeleton.py --outdir book/figs/ch13 --datadir book/data/ch13
```
