# 書籍コード（公開用）

このフォルダは、書籍「Qiskitによる量子計算」の本文で使用したコードを公開するためのものだ。
LaTeX原稿は含めない。

## 構成

- `code/`：各章のコード（本文の実験に対応）
- `CHAPTERS.md`：章とディレクトリ対応表
- `requirements.txt`：依存関係（公開時点で版を固定する）

## 命名ルール

- 章番号は `ch01` のようにゼロ埋めする
- 本文の節番号と対応させたい場合は `secXX_YY_*.py` のように揃える
- 生成物は原則として `out/`（画像）と `data/`（CSV/JSON等）に出す

## 実行の考え方

本書の方針として、コードと本文の一致を最優先する。
そのため各章は「最小で動く」単位に切り、実験結果の解釈とセットで扱う。
